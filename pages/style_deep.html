# Vue style 深度作用选择器

# 有作用域的CSS

当<style>标签增加scoped属性时，它的CSS只作用于当前组件中的元素 通过PostCSS实现

# 全局样式

当<style>标签没有scoped属性时，样式全局生效

# 深度作用选择器

如果希望scoped样式中的一个选择器能够作用得更深，例如影响子组件，可以使用>>>操作符， sass/scss无法正常解析>>>，需要使用/deep/替换>>>
