# Vue页面刷新页面方案 Badge

由于Vue-router只有在路由变化时，才会更新页面，直接通过重新跳转至当前路由无法达到刷新页面的效果。

通过location.reload才能刷新当前页面，但是通过该方法，页面会出现白屏并重新加载所有资源，体验不好。

以下方案可以实现刷新当前页面内容，不需要重新加载资源类似于前端路由切换。

思路： 由于使用vue-router，页面的前端路由切换是通过更新router-view组件的内容实现的。因此，想要实现刷新页面内容， 可以通过更新router-view组件内容实现。通过v-if控制router-view组件是否渲染来实现刷新页面功能。

  1. 控制router-view组件部分：在app.vue中的router-view组件通过v-if绑定一个Vuex中的变量控制是否渲染router-view组件。
<router-view v-if="!sIsHidePage">
</router-view>
  1. 刷新页面部分：将sIsHidePage设置为true，再在nextTick中将sIsHidePage设置为false，实现刷新页面功能。
reloadPage() {
  this.updateIsHidePage(true);
  this.$nextTick(() => {
    this.updateIsHidePage(false);
  });
}
