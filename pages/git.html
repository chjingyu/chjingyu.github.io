# 搭建git服务

  1. 安装git
sudo apt-get install git
  1. 创建一个git用户，用来运行git服务
sudo adduser git
sudo passwd git
  1. 创建证书登录

收集所有需要登录的用户的公钥id_rsa.pub文件，把所有公钥导入到/home/git/.ssh/authorized_keys文件

  1. 初始化Git仓库

选定一个目录作为Git仓库，如：/git/chromeEx.git，在/git目录下输入命令

sudo git init --bare chromeEx.git

修改git仓库的权限为git；

sudo chown -R git:git chromeEx.git

否则在使用时会出现错误

Failed with error: sha1 file '<stdout>' write error:
Invalid argument remote: error: insufficient permission for adding an object to repository 
database ./objects failed to push some refs to 'ssh://git@10.99.201.91:/git/chromeEx.git'
  1. 禁用shell登录

编辑/etc/passwd文件，找到类似下面的一行：

git:x:1000:1000:git,git,,:/home/git:/bin/bash

改为

git:x:1000:1000:git,git,,:/home/git:/bin/git-shell
  1. 克隆远程仓库

通过git clone命令进行克隆远程仓库，可以正常使用git

git clone ssh://git@10.99.201.91:/git/chromeEx.git
